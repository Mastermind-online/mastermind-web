How to play

Mastermind is a classic code-breaking game for two players. The code maker creates a secret code and the code breaker has to guess the code within a limited number of attempts.

The code breaker makes their first guess by selecting a sequence of colored pegs and placing them in a row on their side of the game board.

The code maker then provides feedback to the code breaker in the form of smaller pegs:

- Black peg: Indicates that a guessed peg is both the correct color and in the correct position.

- White peg: Indicates that a guessed peg is the correct color but in the wrong position.

The code breaker has to use this feedback to refine their subsequent guesses until they guess the correct combination, or they run out of attempts.
